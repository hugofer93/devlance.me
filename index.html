Nosotros

DEVLANCE se centra en apoyar y resolver tus problemas con herramientas específicas en lenguajes como Python y JavaScript .

Nuestro equipo tiene mayor experticia en Django, también usamos otras herramientas como Flask o Django Rest Framework para elaborar soluciones eficientes y potentes .

Trabajamos con Base de Datos como: PostgreSQL, Oracle, MySQL, MariaDB, SQLServer, SQLite, entre otras . Adicional también realizamos diseños de bases de datos .

Usamos Git como Sistema de Control de Versiones, para incorporar modificaciones o agregar funcionalidades en lo posterior a tus proyectos .

Para el rol de SysAdmin / DevOps manejamos Servidores Linux con herramientas como Docker o Docker Compose para realizar despliegues con las mejores prácticas; también contamos con experiencia desplegando aplicaciones PHP con NginX .

Apasionados del Desarrollo Web Backend con Python y SysAdmin / DevOps en Amazon Web Services .
